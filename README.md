# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in PHP.

## Setup

1. Install dependencies:

```bash
composer install
```

2. Run solutions:

```bash
# Run all days
composer solve-all
# or
php run.php

# Run a specific day
composer solve -- 1
# or
php run.php 1

# Run with example input
composer solve -- 1 -e
# or
php run.php 1 -e

# Run directly
php src/Day01/Solution.php
```

## Requirements

- PHP 8.1 or higher
- Composer
